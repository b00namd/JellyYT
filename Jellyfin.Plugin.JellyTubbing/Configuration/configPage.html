YouTube Data API

Erforderlich fuer Trending-Kanal und Kanal-Synchronisation. Erstelle einen Key in der Google Cloud Console (YouTube Data API v3 aktivieren).

Google-Konto (Abonnements)

Lade die von Google heruntergeladene client_secret_*.json Datei direkt hoch.
Erstelle OAuth2-Credentials mit Typ TVs und eingeschraenkte Eingabegeraete in der Google Cloud Console. Kein Redirect-URI noetig.

Abonnements synchronisieren

Ordner auf dem Server fuer STRM-/NFO-/Thumbnail-Dateien. Fuge diesen Ordner als Jellyfin-Bibliothek (Typ: Film) hinzu.
Den Sync-Zeitplan konfigurierst du unter Administration → Geplante Aufgaben → JellyTubbing.

Verbinde dein Google-Konto, um Abonnements zu laden.

 

Streaming
