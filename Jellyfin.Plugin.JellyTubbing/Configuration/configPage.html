YouTube Data API

Erforderlich fuer Trending-Kanal und Kanal-Synchronisation. Erstelle einen Key in der Google Cloud Console (YouTube Data API v3 aktivieren).

Google-Konto (Abonnements)

Erstelle OAuth2-Credentials (Typ: Webanwendung) in der Google Cloud Console. Trage als autorisierten Redirect-URI ein:
Basis-URL deines Jellyfin-Servers. Wird fuer OAuth-Redirect und STRM-Dateien verwendet.

Abonnements synchronisieren

Ordner fuer STRM-/NFO-/Thumbnail-Dateien. Fuge diesen Ordner als Jellyfin-Bibliothek (Typ: Film) hinzu.

Verbinde dein Google-Konto, um Abonnements zu laden.

 

Streaming

Leer lassen, wenn yt-dlp im PATH liegt.
