Geplante Downloads

Der geplante Task läuft standardmäßig stündlich. Das Intervall kann unter Dashboard → Geplante Aufgaben → JellyTube → Process YouTube Download Queue angepasst werden.
Jeder Eintrag hat eine URL, ein optionales Verzeichnis, ein maximales Videoalter (Tage, 0 = kein Limit) und die Option, gesehene Videos automatisch zu löschen.
Das Archiv speichert, welche Videos bereits heruntergeladen wurden, damit sie nicht erneut geladen werden. Zurücksetzen, um alle Videos der geplanten Playlists erneut herunterzuladen.
